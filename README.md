**ABAP JSON Architect** is a powerful, browser-based tool designed to automate the conversion of JSON payloads into production-ready SAP ABAP code. It eliminates manual mapping efforts and syntax errors in SAP Integration projects.

### âœ¨ Key Features
- **Instant Conversion:** Transforms complex JSON into ABAP Types (Structures & Tables).
- **Mapping Support:** Generates `/ui2/cl_json` mapping logic automatically.
- **Code Templates:** Provides ready-to-use Web Service / HTTP Client code snippets (Bearer Token, Auth, etc.).
- **Developer Friendly:** Real-time syntax highlighting, VS Code-like bracket matching, and auto `snake_case` conversion.
- **Zero Setup:** Built with pure HTML/CSS/JS. Runs directly in your browser.
- **Customizable:** Supports prefixing (e.g., `GS_`, `TY_`) and Dark/Light themes.
